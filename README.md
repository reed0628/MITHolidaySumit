# 🚢 出勤紀錄表自動生成器 (2026)

這是一個專為公司同事設計的自動化工具，用於將 1-12 月的空白出勤 Excel 檔案自動填入隨機的上下班時間，並支援請假邏輯處理。

## 🌟 功能特點
- **自動填寫**：自動偵測「工作日」，隨機生成 08:50-09:05 與 18:00-18:10 的簽到退時間。
- **請假邏輯**：
  - 早上請假（至 12:00）：自動將簽到時間設為 13:30。
  - 下午請假：自動將簽退時間設為請假起始時間。
  - 全天請假：標註假別並清空時間。
- **格式保留**：直接讀取原始 Excel 範本，保留所有框線、字體與排版。

## 🚀 如何使用
1. 點擊網頁連結（請在此處貼上你的 Streamlit 網址）。
2. 上傳對應月份的 **空白 Excel 範本**。
3. 如果有請假，在下方選單設定 **日期、假別與時間** 並點擊「新增」。
4. 確認無誤後，點擊 **「生成並下載出勤表」**。

## 🛠️ 技術堆棧
- Python 3.x
- Streamlit (網頁界面)
- Openpyxl (Excel 處理)
